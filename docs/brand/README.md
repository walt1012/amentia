# Pith Brand Direction

Pith's current identity direction is a refined lowercase blue `p` on a clean
white rounded-square tile. The baseline blue is `#2F82F3`.

The mark should feel precise, lightweight, local-first, and quietly capable. It
should avoid mascots, generic AI symbols, circuits, chat bubbles, and decorative
extra marks. The useful detail is the subtle inner notch in the `p`, which makes
the shape custom without hurting small-size legibility.
